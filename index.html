Tags frequentemente usadas em conjunto:

<figure> e <figcaption>: Usados para envolver uma imagem ou gráfico com uma legenda.
<ol> e <li>: Usados para criar listas ordenadas.
<ul> e <li>: Usados para criar listas não ordenadas.
<dl>, <dt> e <dd>: Usados para criar listas de definição.

DL: define a lista de definição:

DT: define um termo na lista de definição;

DD: define a descrição correspondente ao termo definido.

dt -> HTML
dd -> HyperText Markup Language - Linguagem de Marcação de Hipertexto
dt -> CSS
dd -> Cascading Style Sheets - Folhas de Estilo em Cascata

<table>, <thead>, <tbody>, <tfoot>, <tr>, <th>, <td>: Usados para criar tabelas.
<form>, <input>, <button>, <select>, <textarea>: Usados para criar formulários.
<details> e <summary>: Usados para criar um elemento de detalhes que pode ser expandido ou recolhido.

Pseudo-classe: São usados para estilizar elementos com base em seu estado ou interação do usuário.

:hover - Estiliza um elemento quando o cursor do mouse está sobre ele.

:active - Estiliza um elemento enquanto ele está sendo clicado pelo usuário.

:focus - Estiliza o elemento quando ele recebe o foco, após ser selecionado pelo usuário.

:focus-within - Estiliza o bloco de elemento quando qualquer um deles ganha foco.
