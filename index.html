Tags de Texto: usadas para formatar e estruturar o texto da página

Tags como <p> para parágrafos, <h1> a <h6> para títulos, <em> ou <i> para enfatizar o texto em italico, <strong> para destacar o texto em negrito, <blockquote> para criar uma citação em bloco, e <span> Esta é uma tag de contêiner genérica que não adiciona significado semântico ao texto, mas é útil para aplicar estilos ou comportamentos específicos a partes específicas do texto.

Tags de Multimídia:

<video> são usadas para incorporar vídeos.

<audio> são usadas para incorporar áudios.

<iframe> são usadas para incorporar conteúdo de outras fontes na página.

Pseudo-classe: São usados para estilizar elementos com base em seu estado ou interação do usuário.

:hover - Estiliza um elemento quando o cursor do mouse está sobre ele.

:active - Estiliza um elemento enquanto ele está sendo clicado pelo usuário.

:focus - Estiliza o elemento quando ele recebe o foco, após ser selecionado pelo usuário.

:focus-within - Estiliza o bloco de elemento quando qualquer um deles ganha foco.
